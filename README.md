# DaMaiProject
大麦界面，实现多种方式网络访问、数据缓存

### 项目描述

    使用RxJava实现数据异步加载、缓存
    MVP搭建项目
    RxJava + DiskLruCache 实现数据异步缓存、加载
    RxJava + GreenDao 实现数据异步缓存、加载

### 使用技术

* RxJava
* Okhttp
* LruCache
* GreenDao
* MVP
* Retrofit
* glide
